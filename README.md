# pacycle

Cycle the default audio output device of pulseaudio.

Usage:

``` sh
pacycle
````

## Installation

``` sh
git clone https://github.com/ghesy/pacycle.git
sudo make install
```

### Uninstall

``` sh
sudo make uninstall
```

### ArchLinux

There is a [package](https://aur.archlinux.org/packages/pacycle) on the AUR:

``` sh
paru -S pacycle
```
